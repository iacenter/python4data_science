Uso de diccionarios

Como en un diccionario convencional, un diccionario en Python permite asociar elementos con algún valor. Estos elementos los identificamos por una llave, de esta manera un diccionario en python guarda pares llave, valor.

Podemos definir un diccionario de la siguiente manera:

diccionario = {'animal' : 'perro', 'edad' : 5, 'juguetes': ['pelota','peluche'] }
