Exploración de los datos

En los ejercicios anteriores aprendimos a inspeccionar el data set obteniendo una parte de los datos para ver su estructura. En algunas ocasiones, solo necesitamos obtener un subconjunto de los datos.

Podemos seleccionar una sola columna usando corchetes [] y el nombre de la columna entre comillas como se muestra a continuación:

df['species'] # Selecciona la columna 'species' de nuestro data set.

df[['sepal_length', 'sepal_width']] # Nótese los dobles corchetes [] para pasar un arreglo
